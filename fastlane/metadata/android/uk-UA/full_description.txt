Погода та радар дощу для будь-якого місцеположення по всьому світу

Дані про погоду надаються через безкоштовний API від Open-Meteo.com і RainViewer.com.

Особливості:
- Поточна погода
- Погодинний прогноз до 16 днів
- Радар дощу
- Віджет із додатковим автоматичним оновленням позиції GPS
- Без реклами

Мінімальні дозволи:
Cirrus вимагає лише мінімальної кількості дозволів, а саме дозволу ІНТЕРНЕТ. Цей дозвіл необхідний для надсилання HTTP-запитів до віддалених серверів для отримання даних про погоду.
Додатково: авторизація для GPS.
